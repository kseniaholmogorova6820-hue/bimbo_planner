# 💖✨ Bimbo Planner ✨💖

> Личный Telegram-бот-планировщик в эстетике «бимбо/принцесса» 🎀💅

Мой первый pet-проект на Python — бот, который помогает планировать неделю и напоминает о делах. Сделан как vibe-coding эксперимент с помощью AI-ассистента Claude (Anthropic).

---

## 🌸 Что умеет

- 💖 **Понимает обычный язык**: можно писать «завтра 18:30 встреча с Олей» или «пятница вечером — кино»
- 🎀 **Поддерживает разные форматы даты**: `сегодня`, `завтра`, `послезавтра`, дни недели (`пн`–`вс`), и обычные даты типа `25.05` или `25.05.2026`
- ⏰ **Гибкое время**: `18:30`, `1830`, `6 вечера`, `утром`, `вечером` — все варианты понимаются
- 🌅 **Быстрые кнопки времени**: если не указали — спросит одним нажатием
- ✨ **Воскресный пинг в 21:00 МСК**: напоминает составить план на неделю
- 💅 **Напоминание за день**: каждый день в 20:00 присылает дела на завтра
- 🦩 **Управление делами**: кнопки «Готово / Отложить / Удалить» под каждым делом

---

## 🛠 Как устроено

- **Язык**: Python 3
- **Библиотека**: [`pyTelegramBotAPI`](https://github.com/eternnoir/pyTelegramBotAPI)
- **Хранение данных**: JSON-файлы рядом со скриптом (`tasks.json`, `state.json`)
- **Хостинг**: Timeweb Cloud (приложение из репозитория)
- **Парсинг ввода**: регулярные выражения + словари для дней недели и частей суток

Архитектура — «один прогон» (one-shot polling): скрипт запускается раз в N минут, забирает накопленные сообщения через `get_updates(offset=...)`, отвечает на них, проверяет напоминания и завершается. Подходит для бесплатных хостингов, где нельзя держать процесс постоянно.

## 🚀 Запуск у себя

### 1. Клонировать репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/bimbo-planner.git
cd bimbo-planner
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Создать бота в Telegram

1. Найди [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`, придумай имя и username
3. Сохрани полученный токен

### 4. Узнать свой Telegram ID

Напиши [@userinfobot](https://t.me/userinfobot) — он пришлёт твой `Id`.

### 5. Создать файл `.env`

В корне проекта создай файл `.env` (не попадёт в git благодаря `.gitignore`):

```
TOKEN=твой_токен_от_BotFather
MY_ID=твой_telegram_id
TZ_OFFSET=3
```

`TZ_OFFSET` — смещение часового пояса от UTC. Для МСК = 3.

### 6. Запустить

```bash
python planner_bot.py
```

Для постоянной работы — настроить запуск по расписанию (на VPS — через cron, на Timeweb Cloud — через тип сервиса «Приложение»).
